Width & Height

Slides 5 & 8

Theory:

  • Total width = left margin + left border + left padding + content width + right padding + right border + right margin
  • Total height = top margin + top border + top padding + content height + bottom padding + bottom border + bottom margin
  • Width and height define the size of the content area

Properties:

width: 450px;
width: 75%;
width: auto;

height: 125px;
height: 50%;
height: auto;

min-width: 450px;
max-width: 600px;
min-height: 120px;
max-height: 160px;

Margin

Slide 9

Theory:

  • Margin clears space outside the border
  • It is transparent
  • Used to create spacing between elements

Properties:

margin-top: .5em;
margin-right: 1em;
margin-bottom: 2em;
margin-left: 1em;

margin: 1em;
margin: .5em 1em 2em 1em;
margin: 0 1em;
margin: .5em 1em 2em;
Content

Padding

Slide 10

Theory:

  • Padding is space inside the border
  • It surrounds the content
  • It is transparent but affects layout size
padding-top: 0;
padding-right: 1em;
padding-bottom: .5em;
padding-left: 1em;

padding: 1em;
padding: 0 1em;
padding: 0 1em .5em;
padding: 0 1em .5em 1em;
Content

Border

Slide 11

Theory:

  • Border surrounds padding and content
  • It can have width, style, and color
  • It is visible unlike margin and padding
border: 2px solid black;
border-top: 2px solid black;
border-right: 4px double blue;

border-width: 1px 2px;
border-style: dashed;
border-color: black gray;

border-radius: 10px;
box-shadow: 3px 3px 4px red;
Content

Background

Slides 13–14

Theory:

  • Background defines color or image behind content
  • Can include gradients
  • Can control repeat, position, and attachment
background: blue;
background-image: url("image.jpg");

background-repeat: repeat-x;
background-position: center top;
background-attachment: fixed;

background: linear-gradient(to right, white, red);
background: linear-gradient(45deg, red, white, blue);
